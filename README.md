# DSC502: Coding
## Regent's University London — 2026/27

This is the course repository for DSC502: Coding. It contains all session and homework notebooks, datasets, and the development environment configuration for the module.

---

## Getting started

1. Click **"Use this template"** → **"Create a new repository"**
2. Name your repository (e.g. `dsc502-yourname`) and set it to **Private**
3. Once created, click the green **Code** button → **Codespaces** tab → **Create codespace on main**
4. Wait for the environment to build — this installs Python 3.11 and all required packages automatically
5. Post your repository URL in your **Blackboard Journal** as your Week 1 anchor entry — this is your submission record for the whole module

---

## How the repo works

- **Week 1** notebooks are pre-loaded in the `week01/` folder
- **Weeks 2–12** notebooks are released weekly on Blackboard — download and upload to the correct folder as the module progresses
- All datasets are in the `data/` folder and are available from day one
- Push your work to GitHub at the end of every session: `git add . && git commit -m "your message" && git push`

## Module philosophy

This module grounds students in the coding fundamentals that make AI-assisted development meaningful. The goal is not to produce programmers in the traditional sense, but developers who understand enough to work with AI deliberately — and to think critically about what they're building, why, and for whom.
